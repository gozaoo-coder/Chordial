## 计划概述
为播放器界面中意图不明确的按钮添加悬浮提示信息框（Tooltip）。

## 需要修改的文件

### 1. PlayerAlbumCard.vue
- **目标按钮**: 更多选项按钮 (`.more-btn`)
- **当前状态**: 无 title 属性
- **修改内容**: 添加 `title="更多选项"` 属性

### 2. PlayerHeader.vue  
- **目标按钮**: 
  - 返回按钮 (`.back-btn`): 添加 `title="返回"`
  - 设置按钮 (`.settings-btn`): 添加 `title="设置"`
- **当前状态**: 无 title 属性

## 实现方式
使用浏览器原生的 `title` 属性实现悬浮提示，与项目中其他按钮保持一致（如 PlayerControls.vue、WindowControls.vue 等已在使用此方式）。

## 修改详情

### PlayerAlbumCard.vue 第21行
```vue
<button class="more-btn" @click="onMore" v-if="showMore" title="更多选项">
```

### PlayerHeader.vue 第3行
```vue
<button class="back-btn" @click="onBack" title="返回">
```

### PlayerHeader.vue 第9行
```vue
<button class="settings-btn" @click="onSettings" title="设置">
```

## 预期效果
当用户将鼠标悬停在这些按钮上时，会显示相应的提示文字，帮助用户理解按钮功能。
