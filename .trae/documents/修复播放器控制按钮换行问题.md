## 问题
播放器页面中，控制按钮区域（PlayerControls + PlayerVolume）在左侧容器（380px宽度）内发生换行，导致布局不美观。

## 失败前提条件检查
1. ✅ 确认 PlayerControls.vue 和 PlayerVolume.vue 文件存在且可编辑
2. ✅ 确认 PlayerView.vue 的样式结构
3. ⚠️ 需要确保修改后按钮在一行显示，不换行

## 修复方案

### 方案1（推荐）：调整 PlayerControls 样式，防止换行
修改 `src/components/player/PlayerControls.vue`：

```css
.player-controls {
  display: flex;
  align-items: center;
  justify-content: flex-start;
  gap: 16px;
  flex-wrap: nowrap;        /* 禁止换行 */
  flex-shrink: 0;           /* 禁止收缩 */
}
```

### 方案2：调整 PlayerView 的控制区域布局
修改 `src/views/PlayerView.vue` 的 `.controls-section`：

```css
.controls-section {
  display: flex;
  align-items: center;
  justify-content: space-between;  /* 改为两端对齐 */
  gap: 16px;
  width: 100%;
  flex-wrap: nowrap;               /* 禁止换行 */
}
```

### 方案3：减小按钮尺寸或间距
在 `PlayerControls.vue` 中：
- 减小按钮间距 `gap: 12px` 或 `gap: 8px`
- 或减小播放按钮尺寸

## 建议执行
采用 **方案1 + 方案2** 组合：
1. 给 PlayerControls 添加 `flex-wrap: nowrap` 和 `flex-shrink: 0`
2. 给 PlayerView 的 controls-section 添加 `flex-wrap: nowrap`

这样可以确保按钮始终在一行显示，不会被挤压换行。
