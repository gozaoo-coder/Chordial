## 问题分析
歌手页面只显示歌手基本信息（名称、歌曲数、专辑数），但下面的专辑列表和歌曲列表为空白。

**根本原因：**
1. 后端 `get_artist_info` 只返回 `album_ids` 和 `track_ids`（ID列表），不包含完整的 `albums` 和 `tracks` 对象
2. 前端模板渲染条件是 `v-if="artist.albums && artist.albums.length > 0"`，但数据中不存在这两个字段

## 修复方案
在 `ArtistDetail.vue` 中，获取歌手信息后，使用现有的后端 API `get_albums_by_ids` 和 `get_tracks_by_ids` 获取完整的专辑和歌曲数据。

## 具体修改
修改文件：`src/views/ArtistDetail.vue`

1. 导入 `getAlbumsByIds` 和 `getTracksByIds` API
2. 在 `onMounted` 中，获取歌手信息后，根据 `albumIds` 和 `trackIds` 调用对应 API 获取完整数据
3. 将获取的数据赋值给 `artist.value.albums` 和 `artist.value.tracks`
