## 目标
完善播放界面，使用 vh/vw 单位实现根据视口大小合理调整宽高、padding、margin 的响应式布局。

## 当前问题分析
1. **PlayerView.vue**: 使用固定 px 值（如 `padding: 40px 60px 60px`、`gap: 80px`）
2. **PlayerAlbumCard.vue**: 封面尺寸固定 `320px`，字体大小固定
3. **PlayerProgress.vue**: 间距和字体使用固定 px
4. **PlayerControls.vue**: 按钮尺寸固定
5. **PlayerVolume.vue**: 布局和尺寸固定
6. **PlayerHeader.vue**: padding 固定

## 优化方案

### 1. PlayerView.vue 主布局优化
- 使用 `vw` 调整左右布局比例
- 使用 `vh/vw` 调整 padding 和 gap
- 左侧固定区域使用 `vmin` 确保在小屏幕上也能正常显示

### 2. PlayerAlbumCard.vue 专辑卡片优化
- 封面尺寸使用 `vmin` 实现正方形且响应式缩放
- 字体大小使用 `clamp()` 限制最小/最大值
- 间距使用 `vh` 调整

### 3. PlayerProgress.vue 进度条优化
- 间距使用 `vh` 调整
- 字体大小使用 `clamp()`

### 4. PlayerControls.vue 控制按钮优化
- 按钮尺寸使用 `vmin` 或 `clamp()`
- 间距使用 `vw` 调整

### 5. PlayerVolume.vue 音量控制优化
- 滑块高度使用 `vh` 相关值
- 按钮尺寸响应式

### 6. PlayerHeader.vue 头部优化
- padding 使用 `vh/vw`
- 按钮尺寸响应式

## 具体数值规划

| 元素 | 当前值 | 优化后 |
|------|--------|--------|
| 主内容区 padding | `40px 60px 60px` | `3vh 4vw 4vh` |
| 左右区域 gap | `80px` | `5vw` |
| 左侧固定宽度 | `380px` | `min(28vw, 380px)` |
| 专辑封面 | `320px` | `min(30vmin, 320px)` |
| 歌曲标题 | `24px` | `clamp(18px, 2.5vh, 28px)` |
| 控制按钮区域 gap | `16px` | `1.5vh` |
| 头部 padding | `16px 24px` | `1.5vh 2vw` |

- 确保大屏幕下（如 1920x1080）布局美观
