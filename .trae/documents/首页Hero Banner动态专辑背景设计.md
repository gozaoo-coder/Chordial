## 设计方案：多专辑封面动态拼接背景

### 核心特性
1. **多专辑封面拼接网格** - 从音乐库随机选取4-9张专辑封面，以网格形式拼接成背景
2. **动态渐变遮罩** - 从底部向上渐变，确保文字可读性
3. **微动效** - 封面网格缓慢平移，营造沉浸感
4. **智能适配** - 专辑不足时自动降级，支持深色/浅色模式

### 视觉层次
```
[专辑封面网格背景 - 动态拼接]
[渐变遮罩层 - 确保文字可读]
[Hero内容 - 标题/按钮/统计]
```

### 技术实现
- 复用 `recentAlbums` 数据，随机选取背景素材
- CSS Grid 创建封面网格 + `object-fit: cover`
- `filter: blur() brightness()` 处理视觉效果
- CSS 动画实现微动效
- 支持 `prefers-reduced-motion` 媒体查询

### 边界处理
- 无专辑数据时降级为现有渐变背景
- 图片懒加载优化性能
