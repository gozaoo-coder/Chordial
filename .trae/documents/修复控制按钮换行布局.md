## 问题
用户希望控制按钮和音量控制分成两行显示，而不是挤在一行。

## 目标布局
```
第一行: [模式] [上一首] [播放] [下一首] [歌词]
第二行: [音量图标] [音量滑块----------------]
```

## 修复方案

修改 `src/views/PlayerView.vue` 的 `.controls-section`：

```css
.controls-section {
  display: flex;
  flex-direction: column;    /* 改为纵向排列 */
  align-items: center;       /* 居中对齐 */
  gap: 16px;
  width: 100%;
}
```

同时需要将 PlayerVolume 的宽度调整为 100% 以填满整行。

修改 `src/components/player/PlayerVolume.vue`：
```css
.player-volume {
  display: flex;
  align-items: center;
  gap: 8px;
  width: 100%;              /* 填满容器 */
}

.volume-slider {
  flex: 1;                  /* 滑块占据剩余空间 */
  height: 20px;
  display: flex;
  align-items: center;
  cursor: pointer;
}
```

这样控制按钮在一行居中，音量控制单独一行且滑块填满宽度。
